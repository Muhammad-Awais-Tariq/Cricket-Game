# Cricket-Game
Its a simple console based python game that includes a toss system the user play against the computer it supports single as well as the tournaments consisting of multiple matches and it  follows easy-to-understand cricket-inspired rules.
---
## Features
- coin toss (Heads / Tails)
- choice to bat or bowl after toss
- play
 - single match
 - Tournament 
- any number of players upto 11
- Custom overs (no upper limit)
- computer opponent with random choices
- Automatic winner declaration 
---
## How the game works
This game is inspired by the hand cricket where both players choose a number between 1 and 6
### Toss Rules
- you can select Heads or Tails
- if toss is won then u can select:
 - Bat
 - Bowl
- if lost computer automatically decidies
---
### Batting & Bowling Rules
- On each bowl:
 - Both players choose a number between 1 and 6
- If the numbers are different:
 - Then batting player score the runs equal to their number
- If the numbers are same:
 - Then batting player loose a wicket
---
## End of an Innings
An innings end when:
- All players are out 
or
- All overs are completed
---
## Match Result
- Player with the higher score wins
- if square is equal then its a draw
---
## Tournament Rules
- Each tournament consists of multiple matches
- Each math winner gets one point
- The player with the more points win the tournament
---
## How to run the game
1. Make sure python is installed
2. Save the file as cricket_game.py
